HTML Embedded Tags Reference

Embedded tags allow you to incorporate external content such as videos, audio, maps, and other media into your HTML documents . This guide covers the most commonly used embedded tags, their attributes, and best practices.

<iframe>

The <iframe> tag is used to embed an HTML document within another HTML document.


<iframe src="https://www.example.com" frameborder="0" width="100%" height="300"></iframe>
Attribute Description
src Specifies the URL of the document to embed.
frameborder Specifies whether to display a border around the iframe.
width Specifies the width of the iframe.
height Specifies the height of the iframe.

Best Practices:

  • Use the <iframe> tag to embed content from trusted sources only.
  • Avoid using iframes to embed large amounts of content, as it can affect page loading times.

<video>

The <video> tag is used to embed video content into an HTML document.


<video src="video.mp4" controls width="100%" height="300"></video>
Attribute Description
src Specifies the URL of the video file.
controls Specifies whether to display video controls.
width Specifies the width of the video.
height Specifies the height of the video.

Best Practices:

  • Use the <video> tag to embed video content in a responsive manner.
  • Provide alternative text for screen readers using the <track> tag.

<audio>

The <audio> tag is used to embed audio content into an HTML document.


<audio src="audio.mp3" controls></audio>
Attribute Description
src Specifies the URL of the audio file.
controls Specifies whether to display audio controls.

Best Practices:

  • Use the <audio> tag to embed audio content in a responsive manner.
  • Provide alternative text for screen readers using the <track> tag.

Conclusion

Embedded tags allow you to incorporate external content into your HTML documents, enhancing user experience and engagement. By following best practices and using the right attributes, you can ensure that your embedded content is accessible, responsive, and secure.
