HTML Link Tags and Attributes

Basic Link Tag

The basic anchor tag (<a>) creates a hyperlink to web pages, files, email addresses, or any other URL.


<a href="https://www.example.com">Visit Example.com</a>

Link Targets

The target attribute specifies where to open the linked document.


<!-- Opens in a new tab/window -->
<a href="https://www.example.com" target="_blank">Open in new tab</a>

<!-- Opens in the same frame (default) -->
<a href="https://www.example.com" target="_self">Open in same tab</a>

<!-- Opens in the parent frame -->
<a href="https://www.example.com" target="_parent">Open in parent frame</a>

<!-- Opens in the full body of the window -->
<a href="https://www.example.com" target="_top">Open in full window</a>

Email Links

Create links that open the user's email client.


<a href="mailto:example@email.com">Send email</a>

<!-- With subject and body -->
<a href="mailto:example@email.com?subject=Hello&body=How%20are%20you">
    Send email with pre -filled subject and body
</a>

Download Links

Create links that prompt the user to download a file.


<a href="path/to/file.zip" download>Download file.zip</a>
