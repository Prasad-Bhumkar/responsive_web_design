Quick Reference

List Type Tags Description
Unordered List <ul>, <li> Creates a bulleted list
Ordered List <ol>, <li> Creates a numbered list
Description List <dl>, <dt>, <dd> Creates a description/definition list

Unordered Lists

Unordered lists are used when the order of items doesn't matter. Each item starts with a bullet point.

Basic Unordered List


<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

Ordered Lists

Ordered lists are used when the order of items matters. Each item starts with a number.

Basic Ordered List


<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>

Description Lists

Description lists are used to describe terms or definitions.

Basic Description List


<dl>
    <dt>Term 1</dt>
    <dd>Definition 1</dd>
    <dt>Term 2</dt>
    <dd>Definition 2</dd>
</dl>

Nested Lists

Nested lists are used to create sublists within a list.

Basic Nested List


<ul>
    <li>Item 1</li>
    <li>Item 2
        <ul>
            <li>Subitem 1</li>
            <li>Subitem 2</li>
        </ul>
    </li>
    <li>Item 3</li>
</ul>

Custom Styling

You can customize the appearance of lists using CSS.

Custom List Styles


ul {
    list-style-type: none;
    padding: 0;
    margin: 0;
}

li {
    padding: 10px;
    border-bottom: 1px solid #ccc;
}

li:last-child {
    border-bottom: none;
}
