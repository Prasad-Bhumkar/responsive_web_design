<table> Tag

The <table> element defines an HTML table.

Example:

<table> <tr> <th>Name</th> <th>Age</th> </tr> <tr> <td>John</td> <td>25</td> </tr> </table>

Attributes:

  • border: specifies the border width
  • cellpadding: specifies the space between the cell content and the cell border
  • cellspacing: specifies the space between cells

<thead> Tag

The <thead> element groups header content in an HTML table.

Example:

<table> <thead> <tr> <th>Name</th> <th>Age</th> </tr> </thead> <tbody> <tr> <td>John</td> <td>25</td> </tr> </tbody> </table>

Best Practices

Use <thead>, <tbody>, and <tfoot> to define the structure of your table.

Use <th> for table headers and <td> for table data cells.

Avoid using tables for layout purposes.

Accessibility Note

Make sure to provide alternative text for screen readers using the <caption> element.

Use header cells (<th>) to define the structure of your table.

Browser Support

HTML table tags are supported in all modern browsers.

However, some older browsers may have limited support for certain attributes.
