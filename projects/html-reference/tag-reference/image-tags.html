HTML Image Tags and Attributes

Basic Image Tag

The basic image tag (<img>) embeds an image into the webpage. The src and alt attributes are required.


<img src="path/to/image.jpg" alt="Description of image">

Image Dimensions

Specify image width and height using pixels or percentage.


<!-- Using pixels -->
<img src="image.jpg" alt="Image" width="300" height="200">

<!-- Using percentage -->
<img src="image.jpg" alt="Image" width="100%" height="auto">

Responsive Images

Use srcset and sizes attributes for responsive images.


<img src="small.jpg"
     srcset="small.jpg 300w,
             medium.jpg 600w,
             large.jpg 900w"
     sizes="(max-width: 500px) 300px,
            (max-width: 900px) 600px,
            900px"
     alt="Responsive image">

Picture Element

The picture element provides more flexibility for responsive images.


<picture>
    <source media="(min-width: 800px)" srcset ="large.jpg">
    <source media="(min-width: 400px)" srcset="medium.jpg">
    <img src="small.jpg" alt="Responsive image">
</picture>
