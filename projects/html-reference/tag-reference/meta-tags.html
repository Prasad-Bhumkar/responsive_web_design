Learn about the different types of HTML meta tags, their uses, and how to implement them for better SEO, social media, and mobile optimization.

Essential Meta Tags

These meta tags are essential for every HTML document.

  • <meta charset="UTF-8">: Specifies the character encoding of the document.
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Controls the zoom level and screen width of the document.
  • <meta name="description" content="Comprehensive guide to HTML meta tags">: Provides a brief description of the document.
  • <meta name="keywords" content="HTML, meta tags, SEO">: Specifies the keywords related to the document.

SEO Meta Tags

These meta tags are used for search engine optimization (SEO).

Basic SEO

  • <meta name="author" content="HTML Reference">: Specifies the author of the document.
  • <meta name="robots" content="index, follow">: Controls how search engines index and follow links on the page.

Advanced SEO

  • <meta name="google-site-verification" content="...">: Verifies ownership of the site for Google Search Console.
  • <meta name="bing-site-verification" content="...">: Verifies ownership of the site for Bing Webmaster Tools.

Social Media Meta Tags

These meta tags are used for social media optimization.

Open Graph

  • <meta property="og:title" content="HTML Meta Tags Reference Guide">: Specifies the title of the page for Open Graph.
  • <meta property="og:description" content="Complete guide to HTML meta tags and their implementation">: Provides a brief description of the page for Open Graph.
  • <meta property="og:image" content="../images/meta-tags-guide.jpg">: Specifies the image to be used for Open Graph.

Twitter Cards

  • <meta name="twitter:card" content="summary_large_image">: Specifies the Twitter Card type.
  • <meta name="twitter:title" content="HTML Meta Tags Reference Guide">: Specifies the title of the page for Twitter Cards.
  • <meta name="twitter:description" content="Complete guide to HTML meta tags and their implementation">: Provides a brief description of the page for Twitter Cards.

Mobile Meta Tags

These meta tags are used for mobile optimization.

  • <meta name="apple-mobile-web-app-capable" content="yes">: Enables standalone mode for iOS devices.
  • <meta name="apple-mobile-web-app-status-bar-style" content="black">: Specifies the status bar style for iOS devices.

Security Meta Tags

These meta tags are used for security purposes.

  • <meta http-equiv="Content-Security-Policy" content="...">: Specifies the Content Security Policy for the page.

Best Practices

Follow these best practices when using HTML meta tags.

  • Use relevant and concise meta tag descriptions.
  • Avoid using duplicate meta tags.
  • Test your meta tags for accuracy and consistency.
